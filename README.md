# visualizations-without-writing-a-line-of-code-using-mosaic-package

Want to plot visualizations without a single line of code ?
Yes!  You can do that with the help of mosaic package , it will use “ggplot2” to  plot visualizations without writing a single line of code.
All you need to do is :
-- Open RStudio and try this code
install.packages("ggplot2")  
install.packages("mosaic")
library(mosaic)
mplot(your_data)

After that choose type of plot you want,
1 for Univariate Analysis and
2 for Bivariate Analysis  

Then pick and choose your plot options (plot type,x-axis, y-axis, color,Facets etc.) with your mouse!
